# MzansiRides

MzansiRides is a South African car rental web app built with React and Vite.
Users can browse available vehicles, compare daily rates, explore rental guides,
book with confidence, and manage their journey from one modern platform.

## Features

- Responsive car rental pages for mobile and desktop
- Vehicle browsing by brand and model
- Rental workflow sections and booking form
- Blog and FAQ sections for customer education
- Contact and privacy policy pages
- Paystack payment integration
